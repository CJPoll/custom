# UI Spec

In @~/dev/beryl/lib/schemas/ui_component.json, we define a json schema that this feature's ui.yaml (the path to which is in context) MUST conform to. In this feature's ui.yaml (the path to which is in context), define the spec for each component that needs to be created or updated; this must conform to the JSON schema defined in ui_component.json as previously mentioned.
